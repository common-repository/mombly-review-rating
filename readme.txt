Mombly Review Rating allows you to add a simple rating to your reviews on your wordpress blog. 
== Description ==

Mombly Review Ratings allow you to easily insert ratings for your reviews on your wordpress blogs. The format to add a rating is simply just [Rating:3.5/5]. You can even use your own pictures for the stars of the ratings. There is also text support for RSS feeds. Do check out the [Mombly Review Rating Plug-in Page](http://mombly.com/wordpress-plug-ins/mombly-review-rating/) as well.


Some features include:

*   Supports multiple ratings on any page
*   Supports text display and image display for ratings
*   Will generate text output for RSS feeds
*   Customizable text output using variables to customize output
*   Customizable images for rating stars

== Installation ==

1. Unzip and upload `mombly_reviewrating` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place tags like [Rating:3.5/5] in your templates
